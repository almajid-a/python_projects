# Cursed Antiques

## Overview
A text-based interactive game where users explore mysterious antiques. Each choice leads to different outcomes, creating a branching experience.

## Features
- Decision-based gameplay
- Multiple outcomes based on user choices
- Branching logic and game flow
- Input validation

## How to Run

```bash
python main.py
```

## Skills Demonstrated
- Control flow and branching logic
- User input handling
- Structuring interactive programs

## Notes
This project focuses on building interactive experiences using core Python concepts.
