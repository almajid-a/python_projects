# Gambling Game

## Overview
A command-line gambling game where users place bets and receive outcomes based on randomized results. The game tracks player balance and simulates a simple betting experience.

## Features
- User betting system
- Balance tracking
- Randomized outcomes
- Input validation for user entries

## How to Run

```bash
python main.py
```

## Skills Demonstrated
- Control flow and logic design
- Functions and modular structure
- Random number generation
- User input handling and validation

## Notes
This project was developed to strengthen core Python fundamentals through a practical, interactive application.
